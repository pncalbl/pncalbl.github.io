Node.js的安装和环境变量的配置

本文最后更新于：3 个月前

1, 下载安装包

去 nodejs官网 下载 找到 node-v12 的下载清单，找到 node-v12.22.1-x64.msi，下载即可。

2，安装

除了要修改安装路径之外， 其他一切默认即可。

3，环境配置

  • 一般默认安装，都会将路径添加系统变量的path里，所以可以通过，node -v 和 npm -v 查看版本号。

  • 一般开始安装的npm 都是6.x 版本， 我们之后可以通过 npm install -g npm 来做npm的全局升级。

  • 之后我们需要在 D:\PNCA\environment\nodejs\node-v12.22.0\ 新建两个包

    • node_cache
    • node_global
    • 因为开始默认是在 C:\Users\Administrator\AppData\Roaming\npm\node_modules 下，安装全局包的，所以我们修改到D盘下。
    • 可以通过 npm root -g 来查看全局包的路径

  • 然后在命令行里运行两条命令

    • npm config set prefix “D:\PNCA\environment\nodejs\node-v12.22.0\node_global”
    • npm config set cache “D:\PNCA\environment\nodejs\node-v12.22.0\node_cache”

  • 找到 D:\PNCA\environment\nodejs\node-v12.22.0\node_modules\npm\ 下的 .npmrc。

    • 修改其内容为：
    • prefix=D:\PNCA\environment\nodejs\node-v12.22.0\node_global
    • cache=D:\PNCA\environment\nodejs\node-v12.22.0\node_cache

  • 修改用户变量

    D:\PNCA\environment\nodejs\node-v12.22.0\node_global

  • 在系统变量里

    添加 NODE_HOME: D:\PNCA\environment\nodejs\node-v12.22.0\node_modules\

4，测试

npm install express -g
